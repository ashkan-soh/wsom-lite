Snappay Orders Report Plugin Plugin Documentaion

Snappay Orders Manager Plugin provides a focused admin report for stores that use the Snappay payment gateway in WooCommerce.
This plugin is an independent reporting tool and is not affiliated with or endorsed by Snappay.

  • ⛾ Free version features:
  • Shows Snappay orders created in the last 7 days (up to 200 latest orders)
  • Displays the total number of Snappay orders in the store
  • Displays the date of the first Snappay order in the store

A separate Premium version with extended reporting and additional features is available outside WordPress.org (distributed separately).
